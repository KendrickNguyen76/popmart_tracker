=====HELP PAGE=====
Here are a list of commands you can run:
**These are all Case Insensitive**

Basic Commands:
HELP - Display the help page
QUIT - Exit the program

Set Commands:
ADD SET - Add a Popmart set
VIEW SET - View a set's information (name, price, figure names)
DELETE SET - Delete a specific set

Figure Commands:
ADD FIGURE - Add a figure to a Popmart set
MARK FIGURE - Mark a figure as collected
VIEW FIGURE - View a figure's information (name, probability, etc.)
DELETE FIGURE - Delete a specific figure

You can quit most of these commands by inputting an invalid set twice
